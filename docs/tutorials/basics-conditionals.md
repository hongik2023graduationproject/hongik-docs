# 조건문

이 튜토리얼에서는 hong-ik의 조건 분기를 배웁니다.

## 기본 조건문

`만약`/`라면` 키워드를 사용합니다:

```hik
정수 나이 = 20

만약 나이 >= 18 라면:
    출력("성인입니다")
```

## if-else

`아니면` 키워드로 조건이 거짓일 때의 코드를 작성합니다:

```hik
정수 점수 = 75

만약 점수 >= 60 라면:
    출력("합격")
아니면:
    출력("불합격")
```

## else-if 체이닝

`아니면 만약`으로 여러 조건을 검사합니다:

<RunCode>

```hik
정수 점수 = 85

만약 점수 >= 90 라면:
    출력("A")
아니면 만약 점수 >= 80 라면:
    출력("B")
아니면 만약 점수 >= 70 라면:
    출력("C")
아니면 만약 점수 >= 60 라면:
    출력("D")
아니면:
    출력("F")
```

</RunCode>

## 복합 조건

논리 연산자로 여러 조건을 결합합니다:

```hik
정수 나이 = 25
논리 학생 = true

만약 나이 >= 18 && 학생 라면:
    출력("성인 학생입니다")
```

```hik
정수 온도 = 35

만약 온도 < 0 || 온도 > 40 라면:
    출력("극한 온도입니다")
아니면:
    출력("정상 범위입니다")
```

## 비교문 (match)

여러 값을 비교할 때는 `비교`를 사용합니다:

<RunCode>

```hik
정수 요일 = 3

비교 요일:
    경우 1:
        출력("월요일")
    경우 2:
        출력("화요일")
    경우 3:
        출력("수요일")
    기본:
        출력("기타")
```

</RunCode>

## 연습 문제

간단한 로그인 시스템을 만들어보세요:

```hik
문자 비밀번호 = "1234"
문자 입력값 = 입력("비밀번호: ")

만약 입력값 == 비밀번호 라면:
    출력("로그인 성공!")
아니면:
    출력("비밀번호가 틀렸습니다")
```
