# 소개

**hong-ik**은 한글을 기반으로 설계된 자체 프로그래밍 언어 프로젝트입니다.

## 특징

- **한글 문법**: 한국어 키워드로 직관적이고 읽기 쉬운 코드 작성
- **이중 실행 엔진**: C++로 구현된 트리워킹 인터프리터와 바이트코드 VM 지원
- **OOP 지원**: 클래스, 상속, 메서드 등 객체 지향 프로그래밍 완벽 지원
- **풍부한 내장 함수**: 31개의 표준 라이브러리 함수 제공
- **안전한 예외 처리**: try/catch를 통한 구조화된 에러 처리
- **모듈 시스템**: 파일 단위의 코드 조직 및 재사용

## 누가 사용할 수 있나?

- **프로그래밍 입문자**: 한글로 작성된 자연스러운 문법으로 학습
- **한국 개발자**: 친숙한 언어로 알고리즘 개발 및 스크립트 작성
- **언어 연구자**: 프로그래밍 언어 구현 및 인터프리터 개발 학습

## 코드 예제

```hik
함수 인사(문자 이름) -> 문자:
    리턴 "안녕하세요, " + 이름 + "님!"

출력(인사("홍길동"))  // 안녕하세요, 홍길동님!
```

더 많은 예제는 [튜토리얼](/tutorials/basics-variables)을 참고하세요.
