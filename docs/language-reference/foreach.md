# forEach 반복문

`각각`/`에서` 키워드를 사용하여 컬렉션의 각 요소를 순회합니다.

## 기본 구문

```hik
각각 타입 변수 컬렉션 에서:
    // 각 요소에 대해 실행할 코드
```

## 배열 순회

```hik
각각 정수 원소 [1, 2, 3] 에서:
    출력(원소)
```

출력:
```
1
2
3
```

## 변수로 순회

```hik
배열 과일들 = ["사과", "배", "포도"]

각각 문자 과일 과일들 에서:
    출력(과일)
```

출력:
```
사과
배
포도
```

## 활용 예제

### 합계 계산

```hik
배열 숫자들 = [10, 20, 30, 40, 50]
정수 합계 = 0

각각 정수 n 숫자들 에서:
    합계 += n

출력(합계)    // 150
```

### 조건부 처리

```hik
배열 점수들 = [85, 92, 78, 95, 60]

각각 정수 점수 점수들 에서:
    만약 점수 >= 90 라면:
        출력("{점수}: 우수")
    아니면:
        출력("{점수}: 보통")
```
