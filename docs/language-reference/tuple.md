# 튜플 (Tuple)

튜플은 변경할 수 없는(immutable) 순서가 있는 값들의 컬렉션입니다.

## 생성

괄호를 사용하여 튜플을 생성합니다:

```hik
(1, 2, 3)
("홍길동", 25, true)
```

## 인덱싱

배열과 같은 방식으로 요소에 접근합니다:

```hik
출력((1, 2, 3)[0])    // 1
출력((1, 2, 3)[1])    // 2
출력((1, 2, 3)[2])    // 3
```

## 활용

여러 값을 그룹으로 묶어서 사용할 때 유용합니다:

```hik
// 좌표 표현
(10, 20)[0]    // x = 10
(10, 20)[1]    // y = 20
```
