# while 반복문

`반복`/`동안` 키워드를 사용하여 조건이 참인 동안 코드를 반복 실행합니다.

## 기본 구문

```hik
반복 조건 동안:
    // 반복 실행할 코드
```

## 예제

```hik
정수 i = 0
반복 i < 5 동안:
    출력(i)
    i += 1
```

출력:
```
0
1
2
3
4
```

## 카운트다운

```hik
정수 n = 5
반복 n > 0 동안:
    출력(n)
    n -= 1
출력("발사!")
```

출력:
```
5
4
3
2
1
발사!
```

## 무한 루프 주의

조건이 항상 참이면 무한 루프가 발생합니다. 반복 변수를 갱신하는 것을 잊지 마세요:

```hik
// 주의: i를 갱신하지 않으면 무한 루프
정수 i = 0
반복 i < 10 동안:
    출력(i)
    i += 1    // 반드시 갱신 필요
```
