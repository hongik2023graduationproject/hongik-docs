# 예외 처리

`시도`/`실패` 키워드를 사용하여 오류를 안전하게 처리합니다.

## 기본 구문

```hik
시도:
    // 오류가 발생할 수 있는 코드
실패 오류변수:
    // 오류 처리 코드
```

## 예제

```hik
시도:
    정수 x = 10 / 0
실패 오류:
    출력(오류)
```

## 활용 예제

### 타입 변환 오류

```hik
시도:
    정수 값 = 정수변환("숫자아님")
실패 오류:
    출력("변환 실패: {오류}")
```

### 파일 읽기 오류

```hik
시도:
    문자 내용 = 파일읽기("없는파일.txt")
    출력(내용)
실패 오류:
    출력("파일을 읽을 수 없습니다: {오류}")
```

### 안전한 프로그램

```hik
함수 안전나눗셈(정수 a, 정수 b) -> 정수:
    시도:
        리턴 a / b
    실패 오류:
        출력("오류: {오류}")
        리턴 0

출력(안전나눗셈(10, 2))    // 5
출력(안전나눗셈(10, 0))    // 오류 메시지 출력 후 0
```
