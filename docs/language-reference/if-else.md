# 조건문 (if/else)

`만약`/`라면`/`아니면` 키워드를 사용하여 조건 분기를 수행합니다.

## 기본 구문

```hik
만약 조건 라면:
    // 조건이 참일 때 실행
```

## if-else

```hik
만약 x > 5 라면:
    출력("크다")
아니면:
    출력("작다")
```

## else-if 체이닝

```hik
정수 점수 = 85

만약 점수 >= 90 라면:
    출력("A 학점")
아니면 만약 점수 >= 80 라면:
    출력("B 학점")
아니면 만약 점수 >= 70 라면:
    출력("C 학점")
아니면:
    출력("F 학점")
```

## 중첩 조건문

```hik
정수 나이 = 20
논리 학생 = true

만약 나이 >= 18 라면:
    만약 학생 라면:
        출력("성인 학생")
    아니면:
        출력("성인")
아니면:
    출력("미성년자")
```

## 복합 조건

논리 연산자를 사용하여 여러 조건을 결합합니다:

```hik
정수 x = 15

만약 x > 10 && x < 20 라면:
    출력("10과 20 사이")

만약 x == 10 || x == 15 라면:
    출력("10 또는 15")
```
