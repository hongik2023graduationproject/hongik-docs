# 사전 (Dictionary)

사전은 키-값 쌍의 컬렉션입니다.

## 선언

```hik
사전 정보 = {"이름": "홍길동", "나이": "25"}
사전 빈사전 = {}
```

## 값 접근

키를 사용하여 값에 접근합니다:

```hik
사전 정보 = {"이름": "홍길동", "나이": "25"}
출력(정보["이름"])    // "홍길동"
출력(정보["나이"])    // "25"
```

## 값 설정

`설정()` 내장 함수로 값을 추가하거나 변경합니다:

```hik
사전 정보 = {"이름": "홍길동"}
설정(정보, "나이", "25")
출력(정보["나이"])    // "25"
```

## 키 목록

`키목록()` 함수로 모든 키를 배열로 가져옵니다:

```hik
사전 정보 = {"이름": "홍길동", "나이": "25"}
배열 키들 = 키목록(정보)
출력(키들)    // ["이름", "나이"]
```

## 요소 삭제

`삭제()` 함수로 키-값 쌍을 제거합니다:

```hik
사전 정보 = {"이름": "홍길동", "나이": "25"}
삭제(정보, "나이")
출력(키목록(정보))    // ["이름"]
```
