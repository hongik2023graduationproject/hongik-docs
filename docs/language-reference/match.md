# 비교문 (match)

`비교`/`경우`/`기본` 키워드를 사용하여 값에 따른 분기를 수행합니다. 다른 언어의 switch/match에 해당합니다.

## 기본 구문

```hik
비교 값:
    경우 패턴1:
        // 패턴1과 일치할 때
    경우 패턴2:
        // 패턴2와 일치할 때
    기본:
        // 어떤 패턴과도 일치하지 않을 때
```

## 예제

```hik
정수 x = 2

비교 x:
    경우 1:
        출력("일")
    경우 2:
        출력("이")
    경우 3:
        출력("삼")
    기본:
        출력("기타")
```

출력:
```
이
```

## 문자열 매칭

```hik
문자 명령 = "시작"

비교 명령:
    경우 "시작":
        출력("프로그램을 시작합니다")
    경우 "정지":
        출력("프로그램을 정지합니다")
    경우 "종료":
        출력("프로그램을 종료합니다")
    기본:
        출력("알 수 없는 명령")
```

## 기본 절

`기본` 절은 어떤 경우에도 일치하지 않을 때 실행됩니다. 선택 사항이지만, 예상치 못한 값을 처리하기 위해 포함하는 것이 좋습니다.
