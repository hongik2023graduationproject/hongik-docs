# 실수 (Float)

실수 타입은 부동소수점 숫자를 나타냅니다.

## 선언

```hik
실수 pi = 3.14159
실수 음수 = -2.5
실수 영 = 0.0
```

## 연산

정수와 동일한 산술 연산을 지원합니다:

```hik
실수 a = 10.5
실수 b = 3.2

출력(a + b)    // 13.7
출력(a - b)    // 7.3
출력(a * b)    // 33.6
출력(a / b)    // 3.28125
```

## 정수와 실수의 혼합 연산

정수와 실수를 함께 연산하면 결과는 실수가 됩니다:

```hik
정수 a = 10
실수 b = 3.5
출력(a + b)    // 13.5
```
