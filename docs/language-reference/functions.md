## 기본 구문

```hik
함수 이름(매개변수들) -> 반환타입:
    // 함수 본문
    리턴 값
```

## 반환값이 없는 함수

반환 타입 없이 선언할 수 있습니다:

```hik
함수 인사하기(문자 이름):
    출력("안녕하세요, " + 이름 + "님!")

인사하기("홍길동")    // 안녕하세요, 홍길동님!
```

## 다중 매개변수

```hik
함수 계산(정수 a, 정수 b, 문자 연산) -> 정수:
    만약 연산 == "더하기" 라면:
        리턴 a + b
    아니면 만약 연산 == "빼기" 라면:
        리턴 a - b
    리턴 0

출력(계산(10, 3, "더하기"))    // 13
출력(계산(10, 3, "빼기"))      // 7
```

## 재귀 함수

함수가 자기 자신을 호출할 수 있습니다:

```hik
함수 팩토리얼(정수 n) -> 정수:
    만약 n <= 1 라면:
        리턴 1
    리턴 n * 팩토리얼(n - 1)

출력(팩토리얼(5))    // 120
```
