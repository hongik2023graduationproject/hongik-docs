# 매개변수 & 반환값

## 타입이 지정된 매개변수

함수 매개변수에는 타입을 지정합니다:

```hik
함수 더하기(정수 a, 정수 b) -> 정수:
    리턴 a + b
```

## 기본 매개변수

매개변수에 기본값을 지정하면 호출 시 생략할 수 있습니다:

```hik
함수 인사(문자 이름, 문자 접미사 = "님") -> 문자:
    리턴 이름 + 접미사

출력(인사("홍길동"))         // "홍길동님"
출력(인사("홍길동", "씨"))   // "홍길동씨"
```

## 반환 타입

`->` 뒤에 반환 타입을 명시합니다:

```hik
함수 제곱(정수 n) -> 정수:
    리턴 n * n

함수 이름가져오기() -> 문자:
    리턴 "홍길동"
```

## 리턴 키워드

`리턴`으로 값을 반환하고 함수를 종료합니다:

```hik
함수 절대값구하기(정수 n) -> 정수:
    만약 n < 0 라면:
        리턴 -n
    리턴 n
```

## 다양한 타입 매개변수

```hik
함수 정보출력(문자 이름, 정수 나이, 실수 키):
    출력("이름: {이름}")
    출력("나이: {나이}")
    출력("키: {키}")

정보출력("홍길동", 25, 175.5)
```
