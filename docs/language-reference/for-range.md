# for-range 반복문

`반복`/`부터`/`까지` 키워드를 사용하여 범위를 순회합니다.

## 기본 구문

```hik
반복 정수 변수 = 시작 부터 끝 까지:
    // 반복 실행할 코드
```

시작 값부터 끝 값 **미만**까지 반복합니다.

## 예제

```hik
반복 정수 i = 0 부터 5 까지:
    출력(i)
```

출력:
```
0
1
2
3
4
```

## 활용 예제

### 합계 계산

```hik
정수 합계 = 0
반복 정수 i = 1 부터 11 까지:
    합계 += i
출력(합계)    // 55 (1부터 10까지의 합)
```

### 구구단

```hik
정수 단 = 5
반복 정수 i = 1 부터 10 까지:
    출력("{단} x {i} = {단 * i}")
```
