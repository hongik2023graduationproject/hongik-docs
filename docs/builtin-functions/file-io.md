# 파일 I/O

파일 읽기/쓰기를 위한 내장 함수입니다.

## 파일읽기(경로)

파일의 내용을 문자열로 읽어 반환합니다.

```hik
문자 내용 = 파일읽기("data.txt")
출력(내용)
```

### 오류 처리

파일이 존재하지 않으면 오류가 발생합니다. `시도`/`실패`로 처리하세요:

```hik
시도:
    문자 내용 = 파일읽기("없는파일.txt")
실패 오류:
    출력("파일을 읽을 수 없습니다")
```

## 파일쓰기(경로, 내용)

문자열을 파일에 씁니다. 파일이 존재하면 덮어씁니다.

```hik
파일쓰기("output.txt", "안녕하세요!")
```

### 활용 예제

```hik
// 데이터 저장
배열 이름들 = ["김철수", "이영희", "박민수"]
문자 내용 = ""

각각 문자 이름 이름들 에서:
    내용 = 내용 + 이름 + "\n"

파일쓰기("이름목록.txt", 내용)

// 데이터 읽기
문자 저장된내용 = 파일읽기("이름목록.txt")
출력(저장된내용)
```

### 파일 복사

```hik
시도:
    문자 원본 = 파일읽기("원본.txt")
    파일쓰기("사본.txt", 원본)
    출력("파일 복사 완료")
실패 오류:
    출력("오류: {오류}")
```
