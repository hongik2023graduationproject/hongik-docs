# I/O 함수

입출력 관련 내장 함수입니다.

## 출력 (print)

콘솔에 값을 출력합니다.

```hik
출력("안녕하세요")
출력(42)
출력(3.14)
출력(true)
```

출력:
```
안녕하세요
42
3.14
true
```

여러 값 출력 (문자열 연결):
```hik
출력("값: " + 문자변환(100))
```

## 입력 (input)

사용자로부터 콘솔 입력을 받습니다.

```hik
문자 입력값 = 입력("이름을 입력하세요: ")
출력("입력한 값: " + 입력값)
```

실행 결과:
```
이름을 입력하세요: 홍길동
입력한 값: 홍길동
```

숫자 입력:
```hik
문자 입력값 = 입력("숫자를 입력하세요: ")
정수 숫자 = 정수변환(입력값)
출력("입력한 숫자: " + 문자변환(숫자))
```

## 실제 사용 예제

### 계산기 프로그램

```hik
출력("=== 간단한 계산기 ===")

문자 input1 = 입력("첫 번째 숫자: ")
정수 num1 = 정수변환(input1)

문자 input2 = 입력("두 번째 숫자: ")
정수 num2 = 정수변환(input2)

정수 합 = num1 + num2
정수 곱 = num1 * num2

출력("합: " + 문자변환(합))
출력("곱: " + 문자변환(곱))
```

### 대화형 프로그램

```hik
반복 논리 계속 = true 동안:
    문자 입력 = 입력("명령 (계산/종료): ")

    만약 입력 == "종료" 라면:
        출력("프로그램을 종료합니다")
        계속 = false
    아니면 만약 입력 == "계산" 라면:
        문자 a = 입력("첫 번째 숫자: ")
        문자 b = 입력("두 번째 숫자: ")
        정수 result = 정수변환(a) + 정수변환(b)
        출력("결과: " + 문자변환(result))
```
