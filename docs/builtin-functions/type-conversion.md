# 타입 변환

값의 타입을 변환하는 내장 함수입니다.

## 타입(값)

값의 타입을 문자열로 반환합니다.

```hik
타입(42)          // "정수"
타입(3.14)        // "실수"
타입("hello")     // "문자"
타입(true)        // "논리"
타입([1, 2])      // "배열"
타입({"a": 1})    // "사전"
```

## 정수변환(값)

값을 정수로 변환합니다.

```hik
정수변환("42")      // 42
정수변환(3.14)      // 3
정수변환("100")     // 100
```

### 변환 실패

변환할 수 없는 값을 전달하면 오류가 발생합니다:

```hik
시도:
    정수변환("숫자아님")
실패 오류:
    출력(오류)    // 변환 오류 메시지
```

## 실수변환(값)

값을 실수로 변환합니다.

```hik
실수변환("3.14")    // 3.14
실수변환(42)        // 42.0
실수변환("2.5")     // 2.5
```

## 문자변환(값)

값을 문자열로 변환합니다.

```hik
문자변환(42)        // "42"
문자변환(3.14)      // "3.14"
문자변환(true)      // "true"
문자변환([1, 2])    // "[1, 2]"
```
